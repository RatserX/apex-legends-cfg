# apex-legends-cfg

#### mod_config
Custom config file

#### mod_keybind
Custom keybind file

#### mod_optimize_X
Custom optimized settings file

#### mod_optimize_X_reset
Custom reset settings file

#### Installation
Put the following files on *Apex\cfg*

    - autoexec.cfg
    - mod_config.cfg
    - mod_keybind.cfg
    - mod_optimize_X.cfg
    - mod_optimize_X_reset.cfg
***
Paste this in X:\Program Files (x86)\Origin Games\Apex\cfg
